Arduino DysonRemote
====================

Dependencies
------------

This project use third-party libraries:

* [RTClib](https://github.com/adafruit/RTClib): Read date time for DS1307
* [L3G4200D](https://github.com/jarzebski/Arduino-L3G4200D): Read temperature on L3G4200D
